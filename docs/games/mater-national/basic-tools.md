# Basic Tools

Here is a list of tools that you'll need for the easiest modding experience:

- [Notepad++](https://notepad-plus-plus.org/downloads/) - Useful for quick edits to plaintext file formats.
- [Visual Studio Code](https://code.visualstudio.com/) - An alternative to Notepad++, useful for more involved edits to sophisticated plaintext file formats as well as source code.
- [HxD](https://mh-nexus.de/en/hxd/) - Useful for viewing/editing to binary file formats such as streaming containers, animation files, and models directly.
- [ImHex](https://github.com/WerWolv/ImHex) - An alternative to HxD, with advanced templating and scripting support.
- [Adobe Photoshop](https://www.adobe.com/products/photoshop.html) - Useful for previewing and editing textures.
- [Paint.NET](https://www.getpaint.net/download.html) - A free alternative to Adobe Photoshop. You'll need its DDS plugin to be able to open/save .DDS files.
- [GIMP](https://www.gimp.org/downloads/) - An alternative to Paint.NET. You'll need its DDS plugin to be able to open/save .DDS files.
- [Noesis](https://www.richwhitehouse.com/index.php?content=inc_projects.php&showproject=91) and [this plugin](https://github.com/leeao/carsraceorama/releases) - Used to preview 3D models and convert them to various different formats.
- [Blender](https://www.blender.org/) and [these plugins](https://github.com/itsmeft24/cars-research/tree/main/animation_data) - Used for creating and previewing `.got` animations.
- [SLTConvert](https://github.com/leeao/carsraceorama/files/11828410/SLTConvert.zip) - Used to compile `.slt` source files into `.xng` files for use in-game.
